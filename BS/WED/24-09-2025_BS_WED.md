## Dateirechte (Linux)

Dateien und Ordner haben **Rechte** für drei Kategorien:

- **Besitzer (user)** – der User, dem die Datei gehört  
- **Gruppe (group)** – die Benutzergruppe, der die Datei gehört  
- **Andere (others)** – alle anderen

Es gibt drei Grundrechte:

- **r (read)** – lesen  
- **w (write)** – schreiben  
- **x (execute)** – ausführen

---

### Beispiel: `drwxr-xr--`

- **d** = Directory (Ordner)  
- **rwx** = Besitzer hat **lesen, schreiben, ausführen**  
- **r-x** = Gruppe hat **lesen, ausführen**, aber kein Schreiben  
- **r--** = Andere dürfen nur lesen

---

### Häufige Kombinationen (aus Tabelle)

| Anforderung | Rechte (Symbolisch) | Oktal |
|-------------|----------------------|-------|
| Nur Besitzer hat Lese- und Schreibrechte | `rw-------` | 600 |
| Besitzer volle Rechte, Gruppe Lese+Ausführung, andere keine Rechte | `rwxr-x---` | 750 |
| Besitzer Lese+Schreibrechte, Gruppe und andere nur Leserechte | `rw-r--r--` | 644 |
| Besitzer und Gruppe haben Lese- und Schreibrechte | `rw-rw----` | 660 |
| Besitzer Lesen+Schreiben, Gruppe Lesen, Andere Lesen | `rw-r--r--` | 644 |
| Besitzer volle Rechte, Gruppe Lese+Ausführung, andere keine Rechte | `rwxr-x---` | 750 |
| Besitzer Lese+Schreiben, Gruppe Leserechte, andere keine Rechte | `rw-r-----` | 640 |
| Besitzer volle Rechte, Gruppe Lese+Ausführung, andere Lesen | `rwxr-xr--` | 754 |
| Besitzer Lese+Schreiben, Gruppe keine Rechte, andere Lesen | `rw----r--` | 604 |
| Besitzer volle Rechte, Gruppe Lese+Schreiben, andere keine Rechte | `rwxrw----` | 760 |
| Besitzer Lese+Schreiben, Gruppe Lese+Ausführung, andere keine Rechte | `rw-r-x---` | 650 |
| Besitzer volle Rechte, Gruppe keine Rechte, andere Lesen | `rwx---r--` | 704 |
| Besitzer Lese+Schreiben, Gruppe Schreibrechte, andere keine Rechte | `rw--w----` | 620 |
| Besitzer volle Rechte, Gruppe Ausführungsrechte, andere keine Rechte | `rwx--x---` | 710 |
| Besitzer Lese+Schreiben, Gruppe Lese+Ausführung, andere Ausführung | `rw-r-x--x` | 651 |
| Besitzer volle Rechte, Gruppe Lese+Schreiben, andere Ausführung | `rwxrw---x` | 761 |
| Besitzer Lese+Schreiben, Gruppe Schreibrechte, andere Ausführung | `rw--w---x` | 621 |

## Wichtige Befehle

### Ordnerstruktur für User erstellen
```bash
mkdir /home/username
```

### User löschen (inkl. Home-Verzeichnis)
```bash
sudo userdel -r alfred
```

### Rechte anzeigen
```bash
ls -l
```

### Rechte ändern mit `chmod`
- `chmod 777 datei` → alle dürfen alles  
- `chmod 444 datei` → alle dürfen nur lesen  
- `chmod 666 datei` → alle dürfen lesen und schreiben, aber nicht ausführen  

Die **Oktalzahlen** setzen sich so zusammen:
- r = 4, w = 2, x = 1  
- Summe ergibt Rechte, z. B.:
  - `7` = rwx (4+2+1)  
  - `6` = rw- (4+2)  
  - `5` = r-x (4+1)  
  - `4` = r-- (nur lesen)

### Besitzer ändern mit `chown`
```bash
sudo chown user:group datei
```
